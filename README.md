# Gestão de Clientes

Este é um projeto de exemplo de um aplicativo CRUD (Create, Read, Update, Delete) usando Flask 3.0.3, Bootstrap 5.3 e uma biblioteca chamada `cru.js`.

## Requisitos

- Python 3.7+
- Flask 3.0.3
- Bootstrap 5.3
